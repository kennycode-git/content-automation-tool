## What's new?

### Video b-roll jobs

You can now create short-form videos with stock footage as b-roll clips, using the new **video clips** workflow.

This is designed for cases where you want moving footage rather than image-based carousel videos. Add a few search terms, preview the clips, choose the ones that fit, then generate the final edit.

**Check out the 'Raindrops' template for one idea of how it can be used.**

### Layered style videos

**Layered videos** now let you combine foreground image searches with background video, which merge together to create a powerful visual.

This is useful for creating more advanced edits where still images sit over atmospheric moving backgrounds. You can search for background videos, choose favourites, select the ones you want, and adjust the blend using opacity controls.

Give this a go especially if you want videos that feel more cinematic or high-quality!

**Check out the 'Stargaze' template for a quick example of how it looks.**

### Philosopher and accent curation

The per-batch style settings now include additional customisation! This means each batch can have its own distinct visual direction. 

Most notably, you can:

1. **Select a philosopher** for that batch, which will add images of that philosopher amongst the image carousel. *Tip:* Add the philosopher's name in the batch title and it will automatically find that philosopher in the list! 

If your philosopher/person isn't in that list, don't worry you can add your own items to the list and upload your own images to be saved forever!

Hit preview images and you can review the selected images and refresh them to find new ones.

2. **New accent images and styles.** Check out the new accent colours *Green* and *Purple*, as well as new colour themes if you haven't yet: *Midnight, Dusk, Mocha, Noir*!

You now have full control over the themes; select one in the per-batch style section, then select 'fine-tune grade' to edit any theme to your liking, and save that custom theme for next time!

## What has improved

### Better preview quality

Preview-first renders now use full-quality staged assets instead of soft preview thumbnails.

This should make the preview much more useful when deciding whether the image selection is good enough before generating the final video.

### Easier re-edit controls

There are now more re-edit options available directly from Recent jobs.

This should make it easier to adjust an existing render, change the style, refresh images, or reuse a previous job without having to rebuild everything from scratch.

### Smoother download processes

From mobile, downloading complete jobs gives the option to download straight into other apps! Works perfectly on iOS: just hit download and options such as TikTok/Instagram will pop up. Let me know if there are any issues on Android etc. with this functionality!

## What to try first

If you are not sure where to start, I would suggest testing these in order:

1. Try the new Video Clips tab and preview a few clips before generating.
2. Try a Layered video with a background video selected.
3. Open Recent jobs and test one of the re-edit options.
4. Try a batch title with a philosopher name and check the philosopher image controls.
