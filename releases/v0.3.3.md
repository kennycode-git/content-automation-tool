# PassiveClip v0.3.3

## New

- Video b-roll jobs help create polished short-form edits from stock footage.
- Layered style videos combine foreground images with looping background motion.
- Philosopher and accent curation is now available in per-batch style settings.

## Improved

- Preview-first renders now use full-quality staged assets instead of soft preview thumbnails.
- Re-edit controls are easier to access from recent jobs.
